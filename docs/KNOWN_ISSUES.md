# 已知问题与版本说明（模板）

本文件用于 **RC / 正式版** 发布时同步记录**用户可见**问题与限制。与 [MVP_LIMITATIONS.md](./MVP_LIMITATIONS.md) 区分：**MVP_LIMITATIONS** 偏能力矩阵（Should/环境）；**KNOWN_ISSUES** 偏具体缺陷与规避方式。

---

## 使用方式

1. 发布前复制本节「版本块」模板，填写版本号与日期。
2. 每条问题含：**现象**、**影响**、**规避/计划**、**跟踪（可选）**。
3. 关闭的问题可移至「已解决」或归档链接。
